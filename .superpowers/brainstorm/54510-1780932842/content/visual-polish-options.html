Visual Polish Improvements

Which improvements would have the biggest impact for you?

A

Smoother Animations

Spring-based transitions when elements move, swap, or appear/disappear. Makes the step-by-step flow feel more alive and less mechanical.

B

Dark/Light Theme Toggle

Let users switch between the current dark theme and a light theme. Better for different lighting conditions and accessibility needs.

C

Color-Coded Operation Types

Use distinct colors for different operation phases: comparing (yellow), swapping (orange), completing (green), current focus (pink). Makes state easier to parse at a glance.

D

Keyboard Navigation + Focus Indicators

Full keyboard control (arrows to step, space to play/pause) with visible focus rings. Helps users navigate without mouse.

E

Better Typography & Spacing

Clearer font hierarchy, more generous spacing, and improved visual rhythm. Reduces cognitive load when reading code and visualization together.
